Rasman is an open-source Web app that runs on the Raspberry Pi and lets you control the BioMonstaaar algae photobioreactor (PBR) from any browser.


Features

  • Web user interface provides full control of the bioreactor
  • Status page shows up-to-date sensor readings and motor status
  • view historical data up to 30 days old (configurable)
  • automatically uploads data to the BioMonstaar portal
  • runs in headless mode.. no keyboard, mouse or monitor needed after initial setup

Tech Details for Geeks

  • small footprint, < 400 lines of Python code.. loads and runs quickly on a Raspberry Pi
  • uses the Flask web microframework
  • interactive plot using Dygraphs
  • sqlite3 database
